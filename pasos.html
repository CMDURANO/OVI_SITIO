Pasos para instalacion

Despues de haber descargado la version que mas se adapte a nuestro S.O, procedemos con la instalacion, asi:

1). Vamos al sitio donde la guardamos y con el segundo boton del mouse le damos Ejecutar como administrador, como muestra la imagen,

2). Nos sale esta ventana y solo le damos Next,

3). En esta parte dejamos como esta por defecto y le damos Next,

4). En esta parte dejamos como esta por defecto y le damos Next,

5). En esta parte cambiamos la que esta marcada por "Use Git from the Windows Command Prompt",

6). Nos muestra la opcion que si deseamos utilizar la consola de comandos que trae Git o la que trae el Sistema Operativo Windows, lo ideal es usar la Git,

7). Dejamos tal como esta y le damos Next,

8). En esta parte no movemos nada, solo le damos Next,

9). Aqui dejamos ls opciones marcado como estan, solo le damos install, esperamos unos minutos para que se instale, todo depende de la maquina que tengamos,

10). Solo basta con darle finish y ya tenemos en nuestro equipo instaldo el Git,
