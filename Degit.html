ACERCA DE GIT

Git es un Sistema de control de versiones, el cual se ocupa de administrar un porgrama o software desarrollado por muchos programadores, que con el paso del tiempo va a crecer de una forma casi incontrolable y es alli que Git se emplea para evitarlo. Provee la forma de controlar los cambios, tomando como una foto del codigo que esta en su proyecto local.

Caracteristicas de Git

1). Su forma de almacenamiento de la informacion la realiza de en archivos, esto con el fin de poder ver y editarlos antes de llevar a la pagina web,

2). Se puede que la mayor parte del proceso de la construccion del programa se realiza en local

3).Git previene cambios de terceros sin que estos no queden registrados en su historial,

Ventajas

* Git tiene un sitema de seguridad arbol, esto impide que se le realice cambios en el codigo fuente

* Esta herramienta es gratuita

Desventajas

*Las personas que desarrollan proyectos bajo este control de versiones, debe realizar un manual de uso, quizas como en todos los programas que se desarrollan, con el fin de evitar errores
